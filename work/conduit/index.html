Integrating Conduit in Houdini

Conduit was the next-generation asset management and version control system used at Blue Sky Studios. By the time I joined the studio, it was already a mature piece of software itself. However, the integration of Conduit in DCCs was still experimental and needed cleaner solutions. One of my main projects at Blue Sky Studios was architecting the Conduit framework in Houdini using the latest production technologies including USD, Solaris and PDG. In the end, a solid foundation with extremely powerful tools were built which all the artists-facing tools were extended from.
