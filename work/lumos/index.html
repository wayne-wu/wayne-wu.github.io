Lumos

Collaborated with: Sebastian Vargas, Sakshi Rathore, Shubham Sharma

Lumos is a 3D co-operative RPG game built using Unreal Engine 5. In this game, enemies are only visible within a lit area. One player controls the knight (third-person view) who is responsible for killing the enemies. The other player controls the mage (top-down view) who is responsible for turning on the lights scattered around the level. Players must work together to kill all the enemies and survive as many waves as possible. The game is built using UE5's network system and is designed to be played over network.
