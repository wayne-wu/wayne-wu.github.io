Neon Dodge

Neon Dodge is a VR multiplayer dodgeball game made with Unity. While we developed and tested the game with HTC Vive, it is built using the OpenXR plug-in and should therefore work with other compatible VR devices. We use UltimateXR for the avatar, controller, and its built-in IK system. Mirror Networking is used for all the networked components. Check out the gameplay below!

Made by: Wayne Wu, Sebastian Vargas, Sakshi Rathore, Shubham Sharma
